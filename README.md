# COM Sniffer

Программа за мониторинг и логване на COM портове с графичен интерфейс.

## Функции

✅ **Избор на COM порт** - Автоматично сканиране и списък на наличните COM портове  
✅ **Конфигурация** - Скорост (baud rate), Data Bits, Parity, Stop Bits  
✅ **Визуализация** - Приемане на данни в реално време  
✅ **HEX и ASCII преглед** - Двойна визуализация на пакетите  
✅ **Логване във файл** - Съхранение на всички получени данни  
✅ **Времеви печати** - Точен запис на времето на всеки пакет  

## Инсталация

### 1. Инсталирайте Python (3.7+)
Преди всичко, уверете се, че имате инсталиран Python 3.7 или по-нова версия.

### 2. Инсталирайте необходимите библиотеки

Отворете Command Prompt или PowerShell в папката на проекта и изпълнете:

```bash
pip install pyserial PyQt5
```

Или ако искате да ползвате virtual environment (препоръчено):

```bash
python -m venv venv
venv\Scripts\activate
pip install pyserial PyQt5
```

## Употреба

### Стартиране на приложението

```bash
python com_sniffer.py
```

Или ако ползите virtual environment:

```bash
venv\Scripts\activate
python com_sniffer.py
```

### Използване на GUI

1. **Избор на COM порт** - Изберете порт от падащото меню
2. **Конфигурация** - Настройте скоростта и други параметри според вашия сензор/устройство
3. **Выбор на лог папка** (опционално) - Кликнете на "Избор на папка за log"
4. **Включване на логване** - Отметнете "Логване във файл" ако искате да спестите данните
5. **Стартиране** - Натиснете "СТАРТ"
6. **Мониторинг** - Данните ще се появяват в реално време
7. **Спиране** - Натиснете "СТОП" когато сте готови

## Параметри

| Параметър | Описание | Типични стойности |
|-----------|---------|------------------|
| **Скорост (baud rate)** | Скорост на комуникация | 9600, 115200, 230400 |
| **Data Bits** | Брой на битовете за данни | 8 (стандартен) |
| **Parity** | Проверка на паритета | NONE (стандартен) |
| **Stop Bits** | Брой на стоп битовете | 1 (стандартен) |

## Лог файлове

Лог файловете се съхраняват в:
- **Избрана папка** - Ако сте избрали специфична папка
- **%USERPROFILE%\COM_Sniffer_Logs** - По подразбиране

Формат на лог файла:
```
=== COM Sniffer Log ===
Начало: 2026-02-04 14:30:45
Порт: COM3, Скорост: 9600 baud
Data Bits: 8, Parity: NONE, Stop Bits: 1
==================================================

[14:30:46.123] HEX: 48 65 6C 6C 6F
[14:30:46.123] ASCII: Hello
```

## Решаване на проблеми

### Грешка: "ModuleNotFoundError: No module named 'PyQt5'"
Решение:
```bash
pip install PyQt5
```

### Грешка: "ModuleNotFoundError: No module named 'serial'"
Решение:
```bash
pip install pyserial
```

### Няма видни COM портове
- Уверете се, че устройството е свързано към компютъра
- Проверете в Device Manager (devmgmt.msc) дали портът е виден
- Преопитайте с кликване на бутона "Обновяване"

### Грешка при отваряне на порт
- Уверете се, че другото приложение не използва същия порт
- Проверете дали сте избрали правилния порт
- Опитайте друга скорост или параметри
